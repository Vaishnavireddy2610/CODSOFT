# 🎮 Tic Tac Toe AI

This is a Tic Tac Toe game built using **Python and Streamlit**.  
It allows you to play against an AI or with another player.

---

## 🚀 Features

- 🤖 Play against AI
- 👥 2 Player mode (same system)
- 🎚 Difficulty levels:
  - Easy
  - Medium
  - Hard
- 🎯 Winning line highlight
- 🏆 Score tracking (updates instantly)
- 🎨 Clean and simple UI
- ⏳ AI thinking delay (feels realistic)

---

## 🧠 AI Logic

The AI uses the **Minimax Algorithm**:

- Easy → random moves  
- Medium → mix of random + smart  
- Hard → always best move

## ⚙️ How to Run

1. Install dependencies:

pip install -r requirements.txt  

2. Run the app:

streamlit run app.py  

3. Open in browser:

http://localhost:8501  

---

## 🎮 How to Play

1. Select mode (AI or 2 Player)  
2. Choose difficulty  
3. Click on boxes to play  
4. First to get 3 in a row wins
